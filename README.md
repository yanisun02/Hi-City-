# Hi City｜让小红书里的城市探索，别再「收藏了就当去过了」

> 把你此刻的心情、曾偷偷点过赞的笔记，和这座城市里正在发生的故事，变成一场真实且值得发一条小红书的城市体验。

Hi City 是一个围绕“小红书城市探索体验”设计的产品 Demo。  
它不是给「同城」换个更好看的页面，而是希望帮你**把收藏夹里吃灰的笔记、点赞过的小店、心里想过但没动身的周末，真正变成一次能出门、能发笔记、能被记住的城市生活**。

---

## 为什么做这个？

你现在是不是也这样：

- 在小红书收藏了一堆“南京必去”“上海咖啡店合集”
- 周末还是瘫在床上刷手机，哪儿也没去
- 想出门，但不想做攻略做得像加班
- 收藏了＝去过了

原来的同城给你一堆附近的帖子，不仅里面充斥着你不喜欢的广告，而且你要自己一条一条刷、自己挑、自己比、自己收藏，然后……没有然后了。

所以我做了 Hi City！。

---

## 它能帮你做什么？

Hi City 把你的**状态、偏好、点赞记录和真实笔记**串起来，让小红书从“内容平台”变成“能帮你出门的生活助理”，打通【线上种草+线下体验】的双通道。

### 1️⃣ 城市选择
不只服务“我现在在哪”，也服务“我将要去哪”。  
本地人、旅行者、刚搬来一座城市的人，都能用。

### 2️⃣ 标签 + 自述
轻轻点一点：上班族 / 老土著 / 内向慢热 / 咖啡自习……  
再写一句话，比如：  
> “周五傍晚在静安寺，想吃饱了再看一场不熬夜的小演出。”

AI 会帮你翻译成**场景词**：雨天独处路线、下班后轻松散步、地铁可达小店连逛……

### 3️⃣ 刷灵感
内容流根据你的状态重新排序。  
你会看到笔记上写着「和你很贴」—— 不是你找内容，是内容来找你。

### 4️⃣ 点赞过 → 直接变成路线
你给一篇笔记点赞，它就会出现在地图上。  
切换到「点赞过」模式，选两个点，AI 自动串成一条 **4 小时的夜游路线**，连停留多久、适合做什么都写好了。

**你的收藏，终于不用再吃灰了。**

### 5️⃣ 找搭子
点一下“想吃汤包”，地图上立刻出现附近的汤包店，  
还会弹出一个弹幕：“附近6人正在找搭子”。  
点一下 say hi，不用尴尬开场。  
这是给 i 人准备的社交。

### 6️⃣ 留印记
走完路线之后，AI 自动帮你写好一篇小红书风格草稿：标题、正文、话题标签全都有。  
你只需要微调，然后发布。

**连发笔记的动力，AI 都帮你准备好了。**
