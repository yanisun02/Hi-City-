# Hi City｜让小红书里的城市探索，更像一次被懂你的旅行灵感击中

Hi City 是一个围绕“小红书城市探索体验”设计的产品 Demo。  
它想解决的问题很简单：现在大家在小红书上找城市攻略、周末去处、旅行灵感时，内容很多，但真正适合自己的路线和决策仍然需要手动筛选、反复收藏、来回比较。

所以我做了 Hi City。

别再「收藏了就当去过了」 Hi City! AI小助手，把你此刻的心情、曾偷偷点过赞的笔记，和这座城市里正在发生的故事，变成一场真实且值得发一条小红书的城市体验吧。

### 3. `Hi_City_Second_Version_ManualAction.html`

这是可手动操作体验的网页版本，并且在这个版本里加入了 AI 小助手。

AI 小助手的加入，是这个 Demo 很重要的一次升级。  
它不只是“多了一个聊天框”，而是改变了原本用户在小红书里寻找城市灵感的路径。

过去用户可能需要：

搜索关键词 → 看大量笔记 → 收藏 → 对比 → 整理路线 → 再做决定

加入 AI 后，体验变成：

说出自己的需求 → AI 理解偏好 → 推荐更适合的城市灵感/路线/行动建议 → 用户更快进入决策

AI 小助手主要承担这些功能：

- 帮用户从大量城市内容里快速提炼重点
- 根据用户偏好生成更个性化的探索建议
- 把碎片化笔记转化成更清晰的行动方案
- 降低“想出去玩但不知道去哪”的决策成本

为什么用 AI？  
因为这个场景的核心难点不是内容不够，而是内容太多、选择太散、用户意图很个性化。  
AI 更适合做理解、归纳和个性化推荐，让小红书从“内容平台”进一步变成“生活灵感助手”。
